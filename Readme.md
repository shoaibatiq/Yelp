## About The Project
This is a bot/scraper that can extract multiple restaurants data from yelp.
following are attributes that are extracted:
- Name
- Category
- Address
- Phone
- Health Score
- Timing
- Rating
- Clamied/Unclaimed
- Reviews	Website
- Yelp Link


### Built With

- Python
- Selenium Webdriver
- BeautifulSoup
- xlsxwriter
- xlrd



<!-- GETTING STARTED -->
## Getting Started

Get a local copy and run the bot from Yelp.ipynb.

### Prerequisites

* Selenium
  ```sh
  pip install selenium
  ```
* BeautifulSoup
  ```sh
  pip install bs4
  ```
* xlrd
  ```sh
  pip install xlrd
  ```
* xlsxwriter
  ```sh
  pip install xlsxwriter
  ```

### Installation

2. Clone the repo
   ```sh
   git clone https://github.com/shoaibatiq/Yelp.git
   ```
